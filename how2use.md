# Installation
To use wdx, download the latest [release](https://github.com/cracky22/wdx/releases) of wdx. The directory contains wdx_extension.crx (unzip via [7zip](https://7-zip.org/), installation via browser > Extensions > Developer mode > Load unpacked). After installing the extension, main.py(c) can be executed with [Python](https://www.python.org/downloads/).
